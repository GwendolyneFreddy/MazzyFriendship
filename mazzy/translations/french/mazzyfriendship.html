Introduction

Un élément dont beaucoup trouvent les PNJ de Bioware dépourvus, c'est la relation plus complexe permise par les mods PNJ modernes - surtout en ce qui concerne les "amitiés", les relations que les PNJ ont avec le PJ lorsqu'ils n'ont pas une romance active.

Ce mod ajoute une série de dialogues pour tout PJ éligible, qui vous permettront d'en apprendre plus sur la vaillante petite-personne, et de vous lier d'amitié avec elle. Mazzy n'est pas la femme la plus bavarde au monde, mais elle a certainement ses avis et les énoncera si elle le juge opportun - de même, si elle a des questions, elle n'hésitera pas à les poser.

Visitez le site web ou le forum pour les dernières mises à jour.

Installation

PC

Mazzy Friendship est un mod WeiDU, compressé via une archive extractable .RAR. Décompressez simplement "mazzy_v2.rar" (WinRAR) dans votre répertoire Baldur's Gate II et exécutez "setup-mazzy.exe".

Pour désinstaller, exécutez à nouveau "setup-mazzy.exe" et sélectionnez l'option de désinstallation.

Compatibilité et ordre d'installation

Le mod Mazzy Friendship devrait être compatible avec n'importe quel mod WeiDU, nous ne pouvons cependant pas tous les tester. Il n'est compatible qu'avec BGII : ToB. Si vous rencontrez une erreur, veuillez contactez K'aeloree (kae@spellholdstudios.net) ou bien postez dans les forums.

Bien qu'il ne soit pas requis pour faire correctement tourner Mazzy Friendship, il est toujours utile d'avoir installé la dernière version du BG2 Fixpack.

Foire Aux Questions

Q : Mazzy Friendship est-il compatible avec le mod XYZ ?
A : Il devrait l'être oui, mais nous n'avons pas testé tous les mods disponibles. Si vous rencontrez une erreur, veuillez contactez K'aeloree (kae@spellholdstudios.net) ou bien postez dans les forums.

Q : Quand débutera l'amitié ?
A : Dans les 15 minutes après que Mazzy ait rejoint le groupe.

Q : Ce mod ne respecte pas *du tout* le personnage.
A : Je suis toujours à la recherche de moyens d'améliorer l'incarnation des personnages dans ce mod, envoyez-moi toute suggestion ou critique (kae@spellholdstudios.net) ou bien postez sur les forums.

Q : As-tu l'intention d'étendre le mod en lui ajoutant d'autres dialogues dans le futur ?
A : Je n'ai malheureusement pas le temps de travailler autant que je le voudrais sur le mod, mais si vous avez des suggestions ou des extensions à proposer, merci de poster sur le forum et je les regarderais dès que possible !

Q : Si je veux commencer une partie à ToB, la partie ToB de l'amitié sera-t-elle accessible ?
A : Je crains que non. Vous devez avoir joué à l'amitié de Shadows of Amn pour obtenir celle de ToB.

Q : Ce mod change-t-il le jeu d'une façon ou d'une autre, hormis les nouveaux dialogues de Mazzy?
A : Normalement non.

Q : Y a-t-il une chance pour que des dialogues initiés par le joueur soient ajoutés dans le futur ?
A : Ce n'est pas prévu pour l'heure, mais si des joueurs sont intéressés je serais ravi d'envisager d'en ajouter lors des versions futures.

Q : Je souhaiterais traduire Mazzy Friendship dans une autre langue. Pourrais-tu TRAifier le mod ?
A : Bien sûr, j'en serai heureux. Si vous souhaitez traduire, envoyez-moi un mot et je verrai ce que je peux faire pour la prochaine version.

Q : Mon dieu, quel con. Pitié arrête de modder.
A : ... nan. :D
